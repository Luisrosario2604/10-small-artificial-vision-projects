## Usage

One with minimum matches
```bash
$ python mediamatcher.py --query=./media/query/LGE.jpg --covers=./media/covers/
```

One without minimum matches
```bash
$ python mediamatcher.py --query=./media/query/Theman.jpg --covers=./media/covers/
```
