## Goals

- Convertir el ejercicio 3 en un Jupyter Notebook

## Requirements

* Python 3.7+

* matplotlib == 3.3.4
* numpy == 1.21.3
* pandas == 1.0.5

How to install all the requirements :
```bash
$ pip install -r requirements.txt
```

## Usage

```bash
$ jupyter notebook compute_stats.ipynb
```
